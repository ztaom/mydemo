imac终端操作命令

新建文件：mkfile [-nv] size[b|k|m|g] filename ...
新建文件夹：mkdir dirname...
转换目录：cd
编辑文件：nano filename...编辑完成后 Ctrl＋X 保存退出

css样式属性选择控制

= Exactly equals,
^= Begins with,
$= Ends with,
~= Contains (with word boundary),
*= Contains (without word boundary)
[class^='icon']{width: 20px; height: 20px; float: left}
.icon_1{background:#eee;}
.icon_2{background:#333;}
