页面内容和结构的语义化

  1. SEO的主要技术点在于做好XHTML
  2. SEO的次要技术点在于关键字的搜索排行
  3. SEO的侧重点在于弱关键字带动强关键字
  4. SEO的必要点是数据静态化以及合理的动态化连接
  5. SEO的落实点在于Portal端

  • 单个页面中h1标签是唯一的。且位置尽可能靠近body
  • 页面的主要内容安排在文档流的前面
  • 提高关键词密度
  • a tag都要加title属性
  • img tag都要加alt属性

页面标题

  1. title = h1 + h2
  2. 关键词要放在靠前位置
  3. 关键词之间用下划线_隔开
  4. 文章名_二级频道名_频道名_站点名称，例如： 
    东奥会计在线
    ...
    新闻中心_东奥会计在线
    ...
    国内新闻_新闻中心_东奥会计在线
    ...
    美国前总统克林顿访问朝鲜_专题_新闻中心_东奥会计在线
    ...
    内蒙古赤峰氨气泄露事故202人接受治疗(图)_各地新闻_新闻中心_东奥会计在线
    ...
