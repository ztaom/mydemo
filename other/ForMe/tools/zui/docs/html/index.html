代码缩进及格式化

  1. div的缩进用2个空格，其他tag不缩进

HTML 标签的使用规则

  1. xhtml 标签一定要小写，并有封闭符
  2. 要使用有语义的标签，尽量少使用无语义标签div和span
  3. a标签都要加title属性，img标签都要加alt属性
  4. 不能采用HTML进行定位、布局(例如用空格、br、hr、center、font等标记)，而应该采用CSS

语义化标签书写原则

  1. Block-Level标签，可以包含Block-Level和Inline-Level标签;
  2. P是块级标签，但不能再嵌套块级标签;
  3. Inline-Level标签，仅可以包含文本或其它Inline-Level标签;
  4. 下面的标签不能包含任何Block-Level标签;
    button按钮
    del被删除的文本
    ins插入的文本
    map图片的热区
    object组件
    script客户端脚本
  5. 以下标签也可以认为是Block-Level标签，但不能单独使用。
    dd定义的描述
    dt定义的标题
    li列表的项
    tbody表格体
    td表格单元格
    tfoot表格脚
    th表格头中的单元格
    thead表格头
    tr表格的行
