ZUI规范指南——css规范——模块化思想

面向对象原则

  1. Separate block's structure and skin
  2. Separate container and content
  3. use consistent sementic styles
  4. minimize selectors
  5. expend objects by applying multiple classes to an element
  6. layout contols width, content controls height

总的思想：

  • 一、容器与内容分类
  • 二、结构与皮肤分类
  • 三、轮廓与背景分类
  • 四、对象与混合物分类

    DONT's：

  1. location dependant styles
  2. specify what tag a class applies
  3. height alignments

模块化词汇：

modulecomponentwidgetplugin
