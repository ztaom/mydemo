ZUI规范指南——类名介绍

类名介绍

类名 对应属性值 使用
no-visited a:link, a:visited {} no-visited用在点击链接文字后文本的链接颜色不变，可用于a或其他元素，如：<a href="" class="no-visited">链接文字</a>链接文字
underline text-decoration:underline; underline使用在文本需要下划线的时候使用，可用于a或其他元素，如：<a href="" class="underline">带下划线的文字</a>链接文字，如果不需要使用下划线，可以使用underline的相反属性no-underline
tip border-bottom: 1px #237ad6 dotted; cursor:help; 只能用于a标签，用于帮助文本上，如：<a href="" class="tip">帮助文字</a>帮助文字
bg-none，bg-blue, white etc background, color bg-xxxx表示背景颜色，white表示文本的颜色，具体参见元素库中的"色系和颜色参照表"
t-c text-align:center 表示文本居中对齐
t-l text-align:left 表示文本居左对齐
t-r text-align:right 表示文本居右对齐
t-t vertical-align:top; 表示文本垂直居顶对齐
t-b vertical-align:bottom; 表示文本垂直居底对齐
t-m vertical-align:middle; 表示文本垂直居中对齐
block display:block 用于将inline元素更改成块元素
inline display:inline-block 将块元素写成内联元素
hidden display:none; 隐藏元素
clear， clearfix clear:both; 主要是清楚浮动
float-left float:left 让元素向左浮动
float-right float:right 让元素向右浮动
float-none float:none 让元素没有浮动
p-absolute position:absolute 让元素成为绝对定位元素
p-relative position:relative 让元素成为相对定位元素
break-word word-break:break-all; word-wrap:break-word; table-layout:fixed; *table-layout:auto; 主要是让整行英文文本折行显示
lh-150 line-height:150 控制文本间的间距，分别为lh-150， lh-160， lh-170， lh-180
bl-solid border:1px solid #ccc 控制边框的样式
bl-dashed border:1px dashed #ccc 控制边框的样式
bl-solid-bottom border-bottom:1px solid #ccc 控制底边框的样式
bl-solid-right border-right:1px solid #ccc 控制右边框的样式
bl-colorBlue border-color:#3774c8 控制边框的颜色，详见元素库中的边框样式
imgborder padding:2px; border:1px solid #68bbed; background:#fff; 控制图片的边框样式和背景，用法：<img src="" class="imgborder"/>
fw-b font-weight:bold 文本加粗
fw-n font-weight:none 文本不加粗
f-11 font-size:11 文本的字体大小，分别有f-11，f-12，f-13，f-14，f-15，f-16，f-18，f-20，f-22，f-24，f-40
p-10 padding:10px 设置模块元素的内边距，还有p-5
p-n padding:0px 让模块的内边距为0
pt-5 padding-top:5px; 模块元素距顶的内边距，主要有pt-2，pt-3，pt-4，pt-5，pt-10，pt-20，pt-30，pt-40
pb-5 padding-bottom:5px; 模块元素距底的内边距，主要有pb-2，pb-3，pb-4，pb-5，pb-10
pl-5 padding-left:5px; 模块元素距左的内边距，主要有pl-5，pl-10
pr-5 padding-right:5px; 模块元素距左的内边距，主要有pr-5，pr-10
m-10 margin:10px 设置模块元素的外边距，还有m-5
m-n margin:0px 让模块的外边距为0
mt-5 margin-top:5px; 模块元素距顶的外边距，主要有mt-2，mt-3，mt-4，mt-5，mt-10，mt-20，mt-30，mt-40
mb-5 margin-bottom:5px; 模块元素距底的外边距，主要有mb-2，mb-3，mb-4，mb-5，mb-10
ml-5 margin-left:5px; 模块元素距左的外边距，主要有ml-5，ml-10
mr-5 margin-right:5px; 模块元素距左的外边距，主要有mr-5，mr-10
