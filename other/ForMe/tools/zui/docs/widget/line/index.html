边框line

  1. 边框样式结构包括：line、 bl-solid = border 、 bl-solid-top = border-top 、 bl-solid-bottom = border-bottom 、 bl-solid-right = border-right；将solid换成dashed则为虚线框
  2. 边框的颜色：bl-colorBlue、bl-colorLightBlue、bl-alphaBlue、bl-colorGreen、bl-colorOrange、bl-colorLightRed、bl-colorCoffee
