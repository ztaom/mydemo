热链接

链接颜色的变化

  1. 链接的正常颜色a:link #000;
  2. 鼠标滑动的颜色a:hover #db0010;
  3. 点击过后的颜色a:active #666;

链接一般样式

  1. 无下划线
    2.       无下划线
  2. 有下划线
    3.       有下划线
  3. 点击过后文字不变色
    4.       点击过后文字不变色
  4. 是否鼠标触及弹出提示层
    5.       是否鼠标触及弹出提示层

3、区分内部链接和外部链接

  1. 内部链接:一般的链接结构；
  2. 外部链接:需在链接的文本后增加修饰图片;

4、链接的交互方式

　　全站使用<base target="_blank">作为链接新窗口打开方式，<base target="_self">为全站本窗口打开，特殊情况对特殊链接做单独设定其打开方式！
