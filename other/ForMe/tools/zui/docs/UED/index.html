测试的原则

  1. 理清层次 每个页面的层次关系应该是很清楚的
  2. 划分区域 把页面划分成明确定义的区域
  3. 明确提示 可点击，可操作的地方，给用户明显的标识
  4. 减少干扰 最大限度降低其他非重要元素产生的干扰
